# Norris-Supermarket-Sales-Dashboard-From-TS-ACADEMY-
This project is an interactive Microsoft Excel Sales Dashboard developed to analyze the sales performance of Norris Supermarket. Data source is from TS Academy. 
As a student of data analysis this was one of the project I did

The dashboard transforms transactional sales data into meaningful business insights, enabling management to monitor sales performance, customer behavior, branch contributions, and product performance through an intuitive and interactive reporting interface.

The dashboard was designed using Excel's Business Intelligence features to support data-driven decision-making and provide a comprehensive overview of key sales metrics.

Project Objectives

The primary objectives of this project were to:

Analyze overall supermarket sales performance.
Monitor monthly sales trends.
Compare sales performance across branches and cities.
Evaluate customer purchasing behavior.
Identify high-performing product lines.
Present business insights through an interactive dashboard.



Dataset

The dashboard is based on supermarket transactional sales data containing information such as:

Sales Amount
Product Line
Branch
City
Customer Type
Quantity Sold
Date of Purchase
Order Value

The dataset enables both descriptive and comparative analysis of supermarket operations.
